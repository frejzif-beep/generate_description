# 🛒 E-commerce Product Description Generator

API для автоматической генерации описаний товаров на основе шаблонов. 
Проект разработан для автоматизации рутинных задач контент-менеджмента в E-commerce.

## 📋 О проекте

Сервис помогает владельцам интернет-магазинов быстро создавать уникальные текстовые описания для карточек товаров на основе их технических характеристик. Генерация происходит по умным шаблонам с вариативностью (синонимы, разные формулировки).

## 🚀 Технологии

- **FastAPI** — современный веб-фреймворк
- **SQLAlchemy 2.0** — ORM для работы с БД (асинхронная версия)
- **PostgreSQL** — база данных
- **Pydantic** — валидация данных
- **Jinja2** — шаблонизатор для генерации текстов
- **Async/Await** — полностью асинхронное приложение

## 📋 Требования

- Python 3.10+
- PostgreSQL 14+
- Виртуальное окружение (рекомендуется)
